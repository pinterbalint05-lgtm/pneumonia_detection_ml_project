# Pneumonia Detection from Chest X‑Rays (CNN / VGG16)

This project builds a convolutional neural network (CNN) in Python to classify chest X‑ray images as **pneumonia** or **normal**. We use transfer learning with **VGG16** and evaluate performance using standard classification metrics.

## Dataset
We use the RSNA Pneumonia Detection Challenge (2018) dataset:

- Source: https://www.rsna.org/artificial-intelligence/ai-image-challenge/rsna-pneumonia-detection-challenge-2018

> Note: The dataset is not included in this repository (too large). Download it separately and place it in the folder structure below.

### Expected data folder structure
Create a `data/` directory in the repository root:

```text
data/
  raw/
    (put downloaded dataset files here)
  processed/
    (generated by preprocessing scripts)
```
